A React + Redux job portal where users can browse jobs, manage profiles, and search positions. Fully deployed with responsive UI and interactive features like modals and alerts.

User authentication and logout with confirmation alerts.

Profile management: personal info, education, experience, skills, and about me.

Job listings with real-time search.

Responsive design for desktop and mobile.

Cosmetic pages and modals for better UX.

Frontend: React, Redux Toolkit, React Router, Tailwind CSS

Backend: json-server 

Alerts/Modals: SweetAlert2

Version Control: Git & GitHub
